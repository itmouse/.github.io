IT行业入门指南(其一)

#

标签（空格分隔）： IT 入门

计算机这个行业，这些年似乎愈见火热。今天的信息时代，确实也是计算在作为主角。国内这一批在互联网上叱诧风云的大佬，或多或少，是当年最早接触互联网的IT人士。
那么，现在入行，晚了么？
且不作功利之心，单看计算机的普及程度，也能知道，即使不必称为IT高手，也应该称为IT老油子，能够在这高速信息时代不掉队。
所以，无论早晚，都应该迎头赶上才是上策，而对于新兴少年，则万能避开IT行业，只怕自己学得晚了，落在人后。
IT入门第一步是什么？
所谓工欲善其事必先利其器，一个称手的计算机，是入门的第一步。
那么，如何选择一个称手的电脑呢？
这个问题困扰了很多入门IT的人士，皆不得其要领。
如果你看见一个geek，他会对你说：“电脑嘛，当然是配置越低越好！这样便无法沉溺于游戏，荒废学习！”这一点也没有错，有志于学习的人，最好选购一台低配置电脑（甚至是树莓派），低到什么级别呢？最好是只能跑个linux系统，不代界面，只是一行行的命令行显示。
但是，这仅仅使用于计算机专业的学生，一个普通人，要想入门，也如此选择，未免太过折腾。
对于一个普通人来说，应该选择在能力范围之内的最好配置。
首先，最好是笔记本，可以不分游戏型、商务型，只要是自己能够负担的最好配置即可。便携，性能强劲，且质量保证，最好售后还特别方便。比如“美帝良心”就是不错的选择。
选择这种电脑的好处，便是可以将大多数的精力放在熟悉计算机上，而不是修理上。
系统最好是windows。
这个世界上最好的黑客，应该都是用的linux，最好的开发者，应该用的是macos,这个世界的IT小白，就得先用着Windows,没有其他原因，上手容易，傻瓜式的鼠标单击，用作入门实在太好不过。要知道，这个世界上，Windows占据半壁江山。无须赘言！
系统最好是前一个版本。
比如现在就不要想着使用Windows10，先使用稳定的Windows7。不要想着去尝试新鲜的系统，新系统往往不定，出现的问题也会很灵异，遇到了，也没有解决的办法，只能盼着新的补丁，而前一版本的系统，流行了很长时间，运行可能也会出错，但有着强大的互联网分享链接，能够借鉴他人经验。
总结一下，如果想要进入IT行业，第一步是入手一台电脑，这台电脑应该遵循这两个原则来确定：
1.选择能负担的配置最好的笔记本电脑。
2.使用稳定版本的Windows系统。
那么，这样就算入门第一步了么？
我还建议你配置一个双肩电脑包，把这个包随身携带，时刻告诫自己，你是一个IT人士，对的，给自己一个心里暗示，让自己相信，自己是一个IT人士，这才是第一步，只有从心里接受了这个设定，才算是入门，否则，终究是过门不入。
当然，这一条不宜执行过长，以一周为宜。
